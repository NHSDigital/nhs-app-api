Some features coming gradually to GP surgeries

Some GP surgeries can currently use all the features of the app. Full use of the app will be rolled out gradually, across England, from December 2018.

By using this, you are helping the NHS

We're currently testing the NHS App. The more people use it, the more we can improve it.

Booking appointments in the app also saves your GP surgery time on administration.

What happens now

We'll check if your GP surgery is using all the features of the app.

If they are

You'll create an NHS login.

If they aren't

You can check your symptoms with the app. But you’ll create an NHS login later, when all features are available.
